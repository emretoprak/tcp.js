tcp.js

tcp.js is a Node.js project making TCP communication easy in node.js. It tries to level out the difference between and a client, making the communication more simple. It also has built-in JSON encoding/decoding.

Introduction

tcp.js is a very simple module for node.js. It has developed because the use of the native net module in node.js, can be cumbersome to use for writing TCP servers and clients. Therefore tcp.js was developed, to make TCP communication very easy, by looking at the well-developed library for WebSocket-communication socket.io, and presenting a similar interface, with custom events and emits.

Built in to tcp.js is JSON encoding and decoding, so that JavaScript objects can be passed seamlessly around from server to client and the other way around. While providing this easy interface, the original properties of the tcp library is still available, giving developers all the need for craftin fast and robust TCP applications.

How to Install

$ npm install tcp.js

How to use

Server

var sockets = require('tcp.js').server(1337);

sockets.on('connection', function (socket) {
  socket.send('news', { hello: 'world'});
  socket.on('myEvent', function (data) {
    console.log(data);
  });
});

Client

var sockets = require('tcp.js').client('127.0.0.1', 1337);

sockets.on('connection', function(socket) {
  socket.on('news', function (data) {
    console.log(data);
    socket.send('myEvent', { my: 'data'});
  });    
});
